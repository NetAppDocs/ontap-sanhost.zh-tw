本內容提供主機和版本的特定最佳實務做法、可快速設定連接ONTAP 到靜態儲存設備的SAN主機。

您的SAN主機和ONTAP 支援功能可存放在內部部署、雲端或兩者。


NOTE: 在雲端環境中、主機通常稱為用戶端。此內容中的所有主機專屬資訊也適用於雲端用戶端。

例如、您可以設定內部部署ONTAP 的支援區、以連線至內部部署的SAN主機、或連線至SAN雲端用戶端。您也可以設定 link:https://docs.netapp.com/us-en/cloud-manager-cloud-volumes-ontap/index.html["Cloud Volumes ONTAP"^] 或 link:https://docs.netapp.com/us-en/cloud-manager-fsx-ontap/index.html["Amazon FSX for ONTAP Sf"^] 連線至內部部署SAN主機或連線至SAN雲端用戶端。

正確的組態對於最佳效能和成功的容錯移轉非常重要。
